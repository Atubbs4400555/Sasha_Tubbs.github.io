Notable Projects

The StarMap

This project was a look into pathfinding algorithms. The user chooses a start point and an end point, then the ship finds a path between them. The unique challenge of this was when i decided to have all of the stars constantly update their positions, menaing that traditional pathfinding algorithms weren't going to work and i would instead need to design an algorithm to work on a constantly changing landscape, which, i am proud to say was achieved.

Boids

This project was a look into mimicking flocking behavoirs through code. The user controls settings in the top left to change what type of flock will be formed by the agents, for example cohesion could be increased to force the agents closer together. I also utilised the Niagara particle system in order to improve the overall presentation fo the portfolio.
